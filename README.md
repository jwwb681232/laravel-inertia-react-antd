# Laravel Inertia React Antd Starter

Use this command to create a new project using this starter:

## Requirement
PHP >= 8.1
Nodejs >= 20.10.0

## Installation

```sh
composer create-project jwwb681232/laravel-inertia-react-antd
```

Install npm packages:
```sh
npm install
```

## Run the project
Run the server:
```sh
php artisan serve
```

Run the frontend bundler:
```
npm run dev
```
